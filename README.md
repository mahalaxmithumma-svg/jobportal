# Job Portal Project 🚀

This is a Django-based Job Portal web application where users can register, login, post jobs, and apply for jobs with resume upload functionality.

---

##  Features

- User Registration & Login
- Admin Dashboard
- Job Posting (Company side)
- Job Browsing (User side)
- Job Application System
- Resume Upload & Management
- User Profile Management
- Applied Jobs Tracking

---

##  Tech Stack

- Python
- Django
- HTML
- CSS
- SQL Database

---

##  Project Structure

- loginapp/ → Main application
- myproject/ → Django settings
- templates/ → HTML files
- static/ → CSS, Images
